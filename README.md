# Global Superstore Sales Analytics Dashboard

## Project Overview

This Power BI dashboard was built using the Global Superstore dataset to analyze sales performance, profitability, customer behavior, and product performance. The objective of this project was to create an interactive business intelligence dashboard using Power BI and DAX while implementing industry-level analytics features.

---

## Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Star Schema

---

## Dashboard Pages

- Executive Overview
- Product Analytics
- Customer Analytics
- Advanced Analytics
- Product Details (Drill Through)

---

## Features Implemented

- KPI Cards
- Interactive Slicers
- Drill Through
- Page Navigation
- Decomposition Tree
- Key Influencers (AI Visual)
- Cross Filtering
- Custom DAX Measures
- Data Modeling using Star Schema
